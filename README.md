## Getting Started

### Step 1 : Clone repository

First we will clone this [vex hugo](https://github.com/yoshixmk/vex-hugo) landing page.

### Step 2 : Start the app

```bash
$ cd cd exampleSite/
$ hugo server --themesDir ../..
```
